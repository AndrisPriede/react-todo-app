# React ToDo Application

A  ToDo application built with React as part of the Web Application Development Laboratory Work.

The user can add tasks, mark them as completed, delete them, and the application also saves all tasks in the browser.


#Features

- Add new tasks  
- Remove tasks  
- Mark tasks as *completed*  
- Tasks are saved in *localStorage* 
- Smooth animations for appearing tasks  
- Progress counter: “X of Y tasks completed”  


#Technologies Used

- **React** (Create React App)
- **JavaScript (ES6+)**
- **CSS (custom styling + animations)**
- **localStorage API**
